# `<a>`

| Property | Value |
| -------- | ----- |
| Mandatory | No |
| Location | Somewhere within `<body>` |
| Must be closed | Yes |

## About
This tag creates a link 

## Arguments
| Argument | Description | Type | Mandatory |
| -------- | ----------- | ---- | --------- |
| `href`   | The location the link to go to | URL | Yes |
| `title`  | Text to display when hovering | Text | No  |

## Usage
```
<a href="[url]">[text]</a>
```
