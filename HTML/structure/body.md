# `<body>`

| Property | Value |
| -------- | ----- |
| Mandatory | Yes |
| Location | Under `<html>` |
| Must be closed | Yes |

## About
This wraps all your page content and enables the usage of the scripts loaded from your `<head>`.

## Usage
```
<html>
    [head content, if applicable]

    <body>
        [body content]
    </body>
</html>
```
